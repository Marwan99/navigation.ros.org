DistanceController
##################

Input Ports
-----------

:distance:

  ====== =======
  Type   Default
  ------ -------
  double  1.0
  ====== =======

  Description
      The distance travelled to trigger an action such as planning a path (m).

:global_frame:

  ====== =======
  Type   Default
  ------ -------
  string "map"
  ====== =======

  Description
    	Reference frame.

:robot_base_frame:

  ====== ===========
  Type   Default
  ------ -----------
  string "base_link"
  ====== ===========

  Description
    	Robot base frame.
